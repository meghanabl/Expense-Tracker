# Expense Tracker Application

## Project Overview
This is a simple Expense Tracker application developed as part of an internship assignment.  
The application allows users to securely log in and manage their daily expenses with full backend integration.

## Features
- User authentication using Firebase Authentication (Email/Password)
- Secure session handling
- Add, view, and delete expenses (CRUD operations)
- Real-time data storage using Firebase Firestore
- Clean and responsive user interface

## Technology Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Firebase
  - Firebase Authentication
  - Firebase Firestore Database

## System Architecture
- The frontend interacts directly with Firebase services.
- Firebase handles authentication, database operations, and secure communication.
- The UI updates dynamically based on real-time database changes.

## How to Run the Project
1. Download or clone the repository.
2. Open the `index.html` file in any modern web browser.
3. Register with email and password.
4. Add and manage expenses.

## Security
- Firebase Authentication for secure login
- Token-based session management
- HTTPS communication by Firebase
